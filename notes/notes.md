# 笔记
记录一些零碎的知识点、遇到的问题、撰写的文章等

每周投入至少20小时，按「2天理论学习+3天项目实践+2天代码重构」的循环推进。
初期可参与Global Game Jam等赛事快速积累经验。建议建立个人作品集网站（推荐使用GitHub Pages），持续记录成长轨迹。
